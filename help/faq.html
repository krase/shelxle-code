Why is this program called ShelXle?

The name ShelXle can be read as SHELXL Editor or you can read there the surnames of the authors SHELdrick and HübschLE. The ending LE is also the Swabian diminutive indicating that this program (writen by a Swabian) is only a tiny cute contribution to the crystallographic field.

How to compile the source code?

  1. Download the latest source code from here.
  2. Unpack it and change in the source directory. 
    tar xvjf shelxle-1.0.*.tar.bz2
cd shelxle-1.0.*/
  3. Run qmake. If Qt3 is installed on your machine qmake might be called 'make-q4' 
    qmake
  4. Run make. 
    make 
sudo make install
  5. That's it!

Is ShelXle phoning HOME?

When you start ShelXle it calls this link to figure out if your version is up to date. If it is out dated you see a pop up like this:



If you say 'No' here you will never be annoyed later and ShelXle does not call any site at startup. ShelXle does never send further information to any site! You can force checking by clicking 'Check for new versions now' in the help menu and then you can change your choice for startup chencking at any time.
